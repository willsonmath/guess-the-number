# Guess The Number

My second Python project, built on my Android phone using Pydroid 3 while learning Python.

## Description

This program is a simple number guessing game played in the terminal. The player selects a difficulty level, then tries to guess a randomly generated number.

The game provides feedback after each guess:

- Too high
- Too low
- Correct

It also counts the number of attempts and allows the player to play again after each round.

## Features

- Three difficulty levels (Easy, Medium, Hard)
- Random number generation
- Input validation
- Attempt counter
- Play Again option
- Case-insensitive user input

## Example

```python
Select level (easy/medium/hard): medium

Guess : 50
Too low

Guess : 75
Too high

Guess : 63

Correct!!

Play again?(yes/no) : no
```
